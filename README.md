# obs-streamelements-core

Core SE.Live OBS plugin.

# Deployment

1. Make sure RELEASE_NOTES.md reflects the release content of the plug-in since previous release to the public (`latest` release group).

2. Once this repository is built, move on to [obs-streamelements-installer]([url](https://github.com/StreamElements/obs-streamelements-installer)), and trigger a manual build on the `master` branch.
