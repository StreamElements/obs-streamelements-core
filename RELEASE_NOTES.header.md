# SE.Live Multistreaming

## Multi-Streaming

- SE.Live enables you to stream their content on multiple platforms simultaneously: Twitch, YouTube (Horizontal + Veritcal/Shorts), X (twitter), Kick, Trovo, TikTok, and you can also stream to Custom RTMP destinations.
- A new dock in OBS enables you to manage your connected platforms, set audio channels per output, and the ability to manage multiple canvases.
- StreamElements canvases support vertical streaming, recording, reusing sources and linking scenes.
- Custom canvases can also be used to dual-stream to platforms like Twitch using native OBS support for for those platforms.
